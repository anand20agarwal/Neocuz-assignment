# Product Catalog API

A future-ready Node.js + Express backend for managing products.

## Features

✅ Add product  
✅ List all products  
✅ Find product by ID  
✅ List unique categories  
✅ Group products by category

## Tech

- Node.js
- Express
- SQLite (easy to swap to PostgreSQL/MySQL)
- UUID
- dotenv

## How to Run
 npm start
